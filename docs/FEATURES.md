# ✨ Features

---

## 📱 WhatsApp Integration

- Conecte múltiplas contas WhatsApp
- Receba mensagens em tempo real
- Envie mensagens e resumos de volta

---

## 🤖 IA & Analysis

- Análise de prioridade automática (IMPORTANTE/URGENTE/NORMAL)
- Extração de contexto (tema, cliente, etc)
- Priorização inteligente

---

## 🎙️ Audio Transcription

- Transcrição de áudio com Whisper
- Fallback automático se confiança baixa
- Suporte a português com vocabulário custom

---

## ⏰ Scheduled Summaries

- "Summi da Hora" (resumo horário)
- Agregação de mensagens importantes
- Envio automático via WhatsApp

---

## 💰 Billing

- Integração com Stripe
- Múltiplos planos (Starter, Professional, Enterprise)
- Gerenciamento de assinatura

---

## 👥 Multi-user

- Admin dashboard
- Gerenciamento de usuários
- Configurações customizáveis por usuário

---

## 📊 Analytics

- Dashboard com estatísticas
- Métricas por instância
- Histórico de mensagens

---

## 🔐 Security

- JWT authentication
- Row Level Security (RLS)
- Encrypted data in transit
- LGPD/GDPR compliant
